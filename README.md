# Лабораторная работа 3


## Цель
- Освоить ввод/вывод, потоковую обработку данных и работу с аргументами командной строки.

## Что реализовано
- Линейная интерполяция по отрезкам.
- Интерполяция Ньютона по скользящему окну фиксированного размера.
- Потоковая обработка: точки читаются из stdin, результаты выдаются по мере готовности.
- Разделение на модули: ввод/вывод отдельно от алгоритмов.
- Единая абстракция движка с полиморфным состоянием для разных алгоритмов.

## Формат ввода/вывода
- Вход: строки вида `x y`, `x;y` или `x\ty`, отсортированы по возрастанию `x`.
- Выход: строки вида `algo: x y`.

## Запуск
Пример линейной интерполяции:
```
cat data.txt | dune exec -- bin/main.exe --linear --step 0.5
```

Пример Ньютона по 4 точкам:
```
cat data.txt | dune exec -- bin/main.exe --newton -n 4 --step 0.5
```

Можно запускать оба алгоритма одновременно:
```
cat data.txt | dune exec -- bin/main.exe --linear --newton -n 4 --step 0.5
```

## Структура проекта
- `lib/io.ml` — чтение/парсинг входного потока и печать результата.
- `lib/engine.ml` — обобщенный движок потоковой интерполяции.
- `lib/linear.ml` — линейная интерполяция.
- `lib/newton.ml` — интерполяция Ньютона.
- `bin/main.ml` — CLI и запуск потока обработки.

## Выводы
Реализован потоковый ввод/вывод и две интерполяции в функциональном стиле с разделением ответственности. Настройка алгоритмов и шага выполняется через аргументы командной строки, результаты выдаются по мере поступления данных.
